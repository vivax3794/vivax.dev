This site is hosted on [Cloudflare Pages](https://pages.cloudflare.com/).
Cloudflare processes request metadata (such as IP addresses) to deliver this site.

For details on how Cloudflare handles data, see their [Privacy Policy](https://www.cloudflare.com/privacypolicy).

## Comments

Blog post comments are powered by [Giscus](https://giscus.app/), which uses [GitHub Discussions](https://docs.github.com/en/discussions) as a backend.
When comments load, your browser connects to `giscus.app` and `github.com`.
Posting a comment requires GitHub authentication, subject to [GitHub's Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement) and [Giscus's Privacy Policy](https://github.com/giscus/giscus/blob/main/PRIVACY-POLICY.md).
